Step 1: Install the Reporter

Add the Lingo Guardian reporter to your React/Next.js app to detect overflow issues in real-time.

Terminal 
npm install lingo-guardian-reporter
Your App (e.g., layout.tsx or App.tsx) 
import { useLingoGuardian } from 'lingo-guardian-reporter';

export default function App() {
  useLingoGuardian({ enable: true });
  return <YourApp />;
}
